# BA Project - Promotion Strategy Engine

## Overview
This project builds a machine learning model to predict profit and recommend optimal promotions based on sales data.

## Features
- Data preprocessing
- Model training & evaluation
- SHAP explainability
- Bias checking
- Scoring pipeline for new data

## Files
- promotion_strategy_engine.ipynb → main analysis
- scoring_script.py → prediction pipeline
- model_metadata.json → model info
- requirements.txt → dependencies

## How to Run
```bash
pip install -r requirements.txt
python scoring_script.py
